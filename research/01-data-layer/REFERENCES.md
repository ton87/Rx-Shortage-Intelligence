# Data Layer — References

## Public APIs

- **openFDA Drug Shortages API**: https://open.fda.gov/apis/drug/drugshortages/
  - Endpoint: `https://api.fda.gov/drug/shortages.json`
  - Search syntax: https://open.fda.gov/apis/query-syntax/
  - No auth needed; rate limit ~40 req/min unkeyed, 240/min with key

- **openFDA Drug Label API**: https://open.fda.gov/apis/drug/label/overview/
  - Endpoint: `https://api.fda.gov/drug/label.json`
  - Section field reference: https://open.fda.gov/apis/drug/label/searchable-fields/

- **RxNorm REST API**: https://lhncbc.nlm.nih.gov/RxNav/APIs/RxNormAPIs.html
  - Endpoint base: `https://rxnav.nlm.nih.gov/REST`
  - `/rxcui.json?name=X` — name to RxCUI

- **RxClass REST API**: https://lhncbc.nlm.nih.gov/RxNav/APIs/RxClassAPIs.html
  - Endpoint base: `https://rxnav.nlm.nih.gov/REST/rxclass`
  - `/class/byRxcui.json?rxcui=X&relaSource=ATC` — RxCUI to class
  - `/classMembers.json?classId=X&relaSource=ATC` — class to members
  - Class systems: ATC, EPC (FDA), MOA, PE, PK — ATC is broadest
